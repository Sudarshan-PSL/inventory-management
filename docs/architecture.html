Factory Inventory Management System

A full-stack demo: a Vue 3 single-page app talking to a Python FastAPI service that serves mock factory data held entirely in memory — no database. This page maps the architecture, the technology stack, and how a request flows end to end.

Four global filters — Time Period, Warehouse, Category, Order Status — are held in the useFilters composable and passed as query params on every request. Raw responses sit in refs (allOrders, inventoryItems); views derive display data via computed properties. Inventory has no time dimension, so it ignores the month filter.
